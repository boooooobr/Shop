# Магазин

Данный бот создан для магазина 'Все на свете'



## Инструкция для админов
Клонируете репозиторий с помощью команды
```bash
git clone https://github.com/boooooobr/Shop.git
```
Заходите в папку Shop затем в папку Basic находите файл main и запускаете 
Заходите в бузу данных Questions смотрите вопрос и затем отвечаете на почту котороя была указана


## Бот будет содержать несколько команд

/question - ответить на часто задоваемые вопросы                                        
/recording - запишет ваш запрос в базу данных и через кокоето время с вами свяжется спецыолист


## Как работает данный бот?
/start команда для привествия а так же расказываетс что делает бот.                             
/info она показывает все доступные команды                                  
/question команда отвечает на самые популярные впоросы                                      
если ваш вашего вопроса нету ведите команда                         
/recording она запишет ваш впорос и с вами свяжется спациолист



## Работа над ботом
config создаю переменую с токеном а так же базу данных                                                      
logic создаю базу даных делаю логигу для записывание вопросов в базу данных                                                 
main сначала имортирую токен и все нужные библеотеки создаю базовые хендлеры после чего начинаю делать хендлер                                          
для часто задоваемых вопросов делаю команду /recording
